# Apple Catcher

A browser game: move a cartoon farmer with the arrow keys and catch falling apples in a basket.

## How to play

1. Open `index.html` in a browser (or serve the folder with any static file server).
2. Press **Start Harvest**.
3. Use **←** and **→** only — the farmer moves horizontally.
4. Catch apples as trees shake and drop them into the basket.

## Rules

- Catch **10 apples** to advance to the next level (10 levels total).
- Each higher level drops **more apples at once** and they fall **faster**.
- Miss **3 apples** and you drop back one level (level 3 → 2, level 2 → 1, and so on).

## Files

- `index.html` — page shell
- `style.css` — orchard-themed UI
- `game.js` — canvas game logic
